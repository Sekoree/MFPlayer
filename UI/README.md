# UI applications

This folder is reserved for **end-user applications** built on the MFPlayer libraries (for example a media player, cue player, or soundboard).

Add new `.csproj` projects here and include them in `MFPlayer.sln` under the **UI** solution folder.

Framework code and reusable libraries live under **`MediaFramework`** in the solution (and remain in their current paths on disk: `Media/`, `Audio/`, `Video/`, `NDI/`, etc.).
